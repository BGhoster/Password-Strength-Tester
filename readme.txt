Password Strength Checker
This is a Python program that checks the strength of a user's password. It looks for weaknesses in the password such as length, number and symbol usage, and also checks against a list of commonly used passwords.

Getting Started
Prerequisites
Python 3.x
Running the program
Open a terminal or command prompt
Navigate to the directory where the program is saved
Run the program with the following command: python password_strength_checker.py
Enter the minimum length for your password
Enter your password

Features
Checks if the password is a commonly used password
Checks the password against requirements (uppercase letters, lowercase letters, numbers, symbols, consecutive letters, consecutive numbers, repeating characters, and keyboard patterns)
Calculates a score based on the password requirements and provides feedback to the user on their password strength
Generates a random password if the score is negative

Limitations
The program does not account for dictionary attacks
The program does not consider the use of common phrases or words as part of the password
